# Anime Tracker 

Anime Tracker is an Android application built using Kotlin and Jetpack Compose.
This app allows users to explore trending, top rated, seasonal, and recommended anime using the Jikan API.

# Features

* Browse Top Anime
* Search Anime
* View Anime Details
* Add Anime to Favorites
* Anime Filtering and Sorting
* Pagination using Paging 3
* Room Database for storing favorites
* Modern UI using Jetpack Compose
